# BitzPrice

> Learn Next.js through a sample Next.js app which whos the current Bitcoin value in USD, GBP and EUR

## Quick Start

```bash
# Install dependencies
npm install

# Serve on localhost:3000
npm run dev

# Build for production
npm run build
```
